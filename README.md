# X (Twitter) Clone UI

A fully responsive **X (formerly Twitter) Clone User Interface** built using **HTML** and **Tailwind CSS**. This project replicates the core layout and visual design of X, including the feed, sidebar navigation, trending section, follow suggestions, post composer, and mobile app-like responsiveness.

## 🚀 Features

* Responsive design for Mobile, Tablet, and Desktop
* X-inspired modern UI
* Left navigation sidebar
* Feed timeline with posts
* Post composer section
* Trending topics panel
* "Who to Follow" suggestions
* Mobile bottom navigation
* Floating action button (FAB)
* Sticky headers and sidebars
* Tailwind CSS utility-first styling
* Material Symbols icons integration

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* Google Material Symbols
* Responsive Web Design
